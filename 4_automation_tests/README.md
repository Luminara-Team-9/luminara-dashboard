# 🤖 성능 자동화 테스트 (Automation Tests)

사람을 대신하여 데카트론 사이트를 탐색하고, 페이지 전환 시 발생하는 성능 지표를 자동으로 측정하는 **QA(품질 보증) 자동화 스크립트 구역**입니다. (Node.js 기반)

**핵심 워크플로우:**
1. **Journeys:** 메인 화면 ➔ 상품 상세(PDP) ➔ 결제까지 이어지는 '다단계 사용자 여정'을 Playwright 봇이 자동으로 주행합니다.
2. **Lighthouse Runner:** 주행 중에 프로그래밍 방식으로 Lighthouse를 실행하여 구간별 로딩 점수를 측정합니다.
3. **AI Agents:** 사이트 디자인이 조금 바뀌어도 봇이 고장 나지 않고 알아서 버튼을 찾는 '자가 치유(Self-healing)' 테스트를 수행합니다.
