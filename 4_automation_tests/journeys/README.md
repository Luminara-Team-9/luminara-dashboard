# 🚗 사용자 여정 주행 봇 (Journeys)

실제 고객이 쇼핑하는 흐름을 똑같이 흉내 내는 **Playwright 자동화 스크립트** 모음입니다.

**무엇이 들어가나요?**
- `main-to-pdp.spec.ts`: 메인 페이지에서 특정 상품을 클릭해서 상세 페이지로 넘어가는 스크립트
- `add-to-cart-checkout.spec.ts`: 장바구니에 담고 결제 페이지까지 넘어가는 스크립트

**왜 필요한가요?**
Lighthouse는 보통 멈춰있는 단일 페이지 하나만 검사합니다. 하지만 진짜 성능 병목은 '페이지를 넘어가는 순간'에 발생하기 때문에, 봇이 직접 흐름을 따라가며(Flow API) 테스트해야 정확한 유저 경험(UX)을 측정할 수 있습니다.
