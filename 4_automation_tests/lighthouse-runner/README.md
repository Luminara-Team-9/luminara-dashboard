# 💡 라이트하우스 자동 측정기 (Lighthouse Runner)

구글 크롬의 성능 측정 도구인 Lighthouse를 브라우저 밖(Node.js 터미널)에서 **프로그래밍 방식으로 제어하고 실행하는 코드**입니다.

**무엇이 들어가나요?**
- `playwright-lighthouse` 라이브러리를 활용한 측정 코드
- 측정된 결과(성능 점수, TBT, LCP 등)를 JSON 형태의 파일로 예쁘게 뽑아내서 저장하는 로직

**역할:**
여기서 측정된 JSON 결과물은 이후 3번 폴더(데이터 & AI 파이프라인)로 넘어가 AI 셜록 홈즈의 분석 단서로 사용됩니다.
