# 🩹 AI 자가 치유 테스트 (AI Agents)

오픈소스 AI 에이전트(ZeroStep 등)를 활용하여, 프론트엔드 코드가 변경되어도 **테스트가 스스로 적응하도록 만드는 똑똑한 테스트 구역**입니다.

**왜 필요한가요?**
보통의 테스트 봇은 개발자가 "파란색 장바구니 버튼(ID: btn-cart)을 눌러!"라고 명령합니다. 만약 프론트엔드 개발자가 버튼 ID를 바꾸면 봇은 고장 나서 에러를 뿜어냅니다.
하지만 AI 에이전트를 도입하면 "화면에서 장바구니처럼 생긴 걸 찾아서 눌러"라고 명령할 수 있어서(Self-healing), UI가 살짝 바뀌어도 테스트 코드를 매번 수정할 필요가 없어집니다.
