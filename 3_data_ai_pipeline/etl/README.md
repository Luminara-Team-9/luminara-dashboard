# 🏭 데이터 정제 공장 (ETL: Extract, Transform, Load)

수집기나 성능 측정 봇이 가져온 날것의 복잡한 JSON 데이터를, AI가 읽기 쉽고 대시보드가 그리기 좋은 형태로 **변환하고 정제하는 파이프라인**입니다.

**무엇이 들어가나요?**
- Pandas와 NumPy를 활용한 데이터 변환 스크립트
- 거대한 용량의 Lighthouse JSON 보고서에서 핵심 지표(LCP, TBT, CLS 등)만 쏙쏙 뽑아내는 추출기

**왜 필요한가요?**
Lighthouse가 뱉어내는 원본 JSON 파일은 수만 줄에 달할 정도로 복잡합니다. 이걸 그대로 AI나 프론트엔드에 던져주면 터져버리거나 처리 시간이 너무 오래 걸리므로, 여기서 깔끔한 표(데이터프레임) 형태로 가공해 주어야 합니다.
