# 🕵️‍♂️ AI 분석가 (AI Analyzer)

이 프로젝트의 핵심인 **오픈소스 LLM(DeepSeek-R1) 통합 및 프롬프트 엔지니어링** 구역입니다.

**무엇이 들어가나요?**
- Ollama를 통해 로컬 GPU 서버에서 구동되는 DeepSeek 모델 연동 스크립트
- 정제된 성능 데이터를 AI에게 주입하여 "왜 느려졌는지, 어디 코드를 고쳐야 하는지" 묻는 프롬프트(명령어) 템플릿
- AI가 생성한 '최적화 해결책(Fix Plans)'을 백엔드 DB로 넘겨주는 연결 로직

**목표:**
단순히 "LCP가 3초입니다"라고 알려주는 것을 넘어, AI가 "메인 화면 3번째 이미지 용량이 너무 큽니다. WebP로 변환하세요"라고 **구체적인 행동 지침**을 도출해 내는 것이 목표입니다.
