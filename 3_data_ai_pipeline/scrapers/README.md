# 🕷️ 데이터 수집기 (Scrapers)

국내 이커머스 경쟁사(나이키, 아디다스, 쿠팡 등)의 웹사이트에 주기적으로 접속해서 성능 데이터를 긁어오는 **크롤링 봇**들을 모아둔 폴더입니다.

**무엇이 들어가나요?**
- Python Selenium 또는 Playwright를 활용한 웹 자동화 스크립트
- 주기적으로 대상 사이트를 방문해 화면 로딩 속도나 네트워크 지연 상태를 측정하는 로직

**역할:**
대시보드(1번 폴더)에서 데카트론과 경쟁사를 비교하려면 남의 데이터가 필요합니다. 이 봇들이 밖으로 나가서 그 비교용 기초 식량을 구해오는 역할을 합니다.
