# 📄 완성된 화면 (Pages Layer)

사용자가 URL을 치고 들어왔을 때 보게 되는 **최종 조립된 하나의 화면(페이지)**들을 모아두는 곳입니다.

**우리 프로젝트 적용 예시:**
- `HomePage`: 대시보드 메인 화면 (전체 성능 요약)
- `CompetitorComparePage`: 데카트론 vs 경쟁사(나이키, 아디다스 등) 성능 비교 화면
- `AiReportPage`: AI가 분석한 병목 현상 및 최적화 로드맵 결과 화면

**규칙:**
여기서는 직접 복잡한 로직을 짜지 않습니다. 아랫단(`widgets`, `features`)에 있는 큼직한 부품들을 가져와서 화면의 제 위치에 '배치'만 하는 역할을 합니다.
