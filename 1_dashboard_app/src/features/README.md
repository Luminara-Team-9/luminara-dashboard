# 🚀 사용자 액션 (Features Layer)

사용자가 클릭하거나 입력해서 발생하는 **구체적인 비즈니스 로직(기능)**을 모아두는 곳입니다.

**우리 프로젝트 적용 예시:**
- `DownloadPdfReport`: 분석 결과를 PDF로 다운로드하는 버튼과 그 기능
- `ApplyAiFix`: AI가 제안한 최적화 코드(WebP 변환 등)를 실험실에 적용해보는 버튼
- `FilterByRegion`: '서울', '부산' 등 지역별 지연시간을 조회하기 위해 드롭다운을 선택하는 기능

**규칙:**
버튼이나 폼(Form) 같이 사용자와 직접 '상호작용'하는 녀석들이 여기에 들어갑니다.
