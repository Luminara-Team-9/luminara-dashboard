# 🧬 데이터 모델 (Entities Layer)

우리 프로젝트의 **핵심 비즈니스 데이터(개념)**를 화면에 보여주는 가장 기초적인 단위입니다.

**우리 프로젝트 적용 예시:**
- `Competitor`: '경쟁사'라는 개념 자체 (경쟁사 로고, 이름, 기본 정보 카드)
- `ScoreCard`: 'Lighthouse 점수'라는 개념 자체 (초록색 90점, 빨간색 40점 등 점수를 보여주는 동그란 UI)

**규칙:**
이곳의 코드들은 자기가 어떤 맥락(어느 페이지, 어떤 위젯)에서 쓰이는지 모른 채, 오직 '자기 자신의 데이터'를 보여주는 데만 집중합니다.
