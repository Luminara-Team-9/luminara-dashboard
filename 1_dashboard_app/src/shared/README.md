# 🧰 대시보드 전용 부품 (Shared Layer)

FSD 아키텍처의 가장 밑바닥 계층으로, **'대시보드 앱 내부에서만'** 공통으로 쓰이는 자잘한 도구와 부품들입니다.

**무엇이 들어가나요?**
- 차트 그리는 라이브러리(Recharts 등) 래퍼(Wrapper)
- 대시보드 안에서만 쓰이는 특수한 아이콘들
- 대시보드 내부 API 통신용 커스텀 훅(Hook)

**⚠️ 주의 (`0_shared_packages`와의 차이점)**
최상위의 `0_shared_packages`는 원본/개선 사이트까지 3개 프로젝트가 **모두** 쓰는 찐 공통 자원이고, 이 폴더는 오직 **대시보드 1개** 안에서만 돌려쓰는 녀석들입니다!
