# 🧩 독립적인 큼직한 덩어리 (Widgets Layer)

페이지를 구성하는 **가장 크고 독립적인 UI 블록**들입니다. 여러 개의 기능(`features`)과 데이터(`entities`)가 결합된 완성형 부품입니다.

**우리 프로젝트 적용 예시:**
- `PerformanceChartWidget`: Lighthouse 성능 지표(LCP, CLS 등)를 막대그래프로 보여주는 거대한 차트 영역
- `AiChatWidget`: 하단에 떠 있는 AI 셜록 홈즈와의 채팅창 전체 영역
- `DashboardHeader`: 상단 로고, 로그인 버튼, 메뉴가 다 포함된 헤더 블록

**규칙:**
`Pages` 폴더에서 이 위젯들을 가져다 화면을 구성합니다.
