# 🛠️ 공통 도구 상자 (Utils)

특정 화면이나 사이트에 종속되지 않고, 프로젝트 전반에서 두루 쓰이는 **유용한 로직과 계산 함수**들을 모아둡니다.

**무엇이 들어가나요?**
- `2345ms`를 `2.34초`로 예쁘게 변환해 주는 포맷 변환 함수
- 백엔드 서버(API)에 통신을 요청하는 공통 세팅 (Axios/Fetch 클라이언트)
- 성능 점수 비교를 위한 간단한 통계 로직

**우리 프로젝트 적용 예시:**
- **시간 포맷 변환기:** 자동화 측정 봇이 가져온 `2345ms`라는 데이터를 대시보드 화면에 `2.34초`로 예쁘게 바꿔주는 함수 (`formatTime()`).
- **API 통신 클라이언트:** 프론트엔드가 백엔드 API(5번 폴더)에 "경쟁사 벤치마크 결과 주세요!"라고 요청할 때 사용하는 공통 통신 세팅 코드.

**사용 방법:**
데이터를 가공하거나 서버와 통신해야 할 때, 직접 코드를 새로 짜지 않고 이곳의 도구를 `import` 하여 사용합니다.
