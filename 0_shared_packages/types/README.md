# 📝 공통 데이터 규격 (Types)

프론트엔드와 백엔드가 데이터를 주고받을 때 서로 오해가 없도록 정의한 **TypeScript 타입(인터페이스) 계약서** 모음입니다.

**무엇이 들어가나요?**
- Lighthouse 측정 결과 JSON 스키마 (`LCP`, `CLS` 등)
- 서버에서 보내주는 통신사/지역별 지연시간 데이터 구조
- API 응답 성공/실패 공통 규격

**사용 방법:**
백엔드에서 데이터를 받을 때나 프론트엔드에서 데이터를 화면에 뿌려줄 때, 이곳에 정의된 타입을 가져다 사용하여 에러를 사전에 방지합니다.
