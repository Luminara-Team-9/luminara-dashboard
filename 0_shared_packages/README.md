# 📦 공용 부품 창고 (Shared Packages)

이 폴더는 데카트론 웹 성능 최적화 프로젝트의 3개 프론트엔드 환경(대시보드, 원본 복제본, 개선 복제본)이 공통으로 사용하는 자원들을 모아둔 중앙 저장소입니다.

**목적:** 
동일한 코드를 여러 프로젝트에 중복해서 작성하는 헛수고를 막고, 데이터 규격과 디자인의 통일성을 완벽하게 유지하기 위함입니다.

**디렉토리 구조:**
- `/types`: 공통 데이터 규격 (설계도)
- `/ui`: 공통 디자인 컴포넌트 (레고 블록)
- `/utils`: 공통 유틸리티 함수 (도구 상자)
- `/config`: 공통 환경 설정 (코딩 규칙 책)
