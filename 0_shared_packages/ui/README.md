# 🎨 공통 디자인 부품 (UI)

원본 데카트론 사이트와 최적화된 사이트가 시각적으로 100% 동일하게 보이도록, 공통으로 사용하는 **UI 컴포넌트(레고 블록)**를 모아둔 곳입니다.

**무엇이 들어가나요?**
- 데카트론 브랜드 컬러 코드 및 폰트 설정
- 사이트 공통 헤더(Header), 푸터(Footer)
- 장바구니 버튼, 텍스트 입력창 등 재사용 가능한 UI 부품

**우리 프로젝트 적용 예시:**
- **데카트론 공식 버튼/헤더:** 상품 장바구니 버튼이나 상단 내비게이션 바를 여기에 한 번만 만들어 둡니다.
- **왜 필요한가요?:** 원본 사이트와 최적화 사이트의 디자인이 조금이라도 다르면 "순수하게 로직(WebP, Lazy Load 등)을 바꿔서 성능이 좋아진 건지" 증명하기 어렵습니다. 이 공용 레고 블록을 두 사이트가 같이 가져다 쓰면 디자인이 완벽히 똑같이 유지됩니다.

**왜 필요한가요?**
버튼 디자인이나 색상이 바뀔 때, 여러 프로젝트를 돌아다니며 수정할 필요 없이 이곳의 코드 한 줄만 수정하면 모든 사이트에 자동 반영됩니다.
