# 🛒 데카트론 실험실 (Digital Twins)

실제 데카트론 라이브 서버에 영향을 주지 않고 웹 성능 최적화 실험(A/B 테스트)을 진행하기 위해 구축한 **디지털 복제본(Digital Twin)** 환경입니다.

**구조 및 목적:**
- `/baseline-site`: 현재 서비스 중인 데카트론 코리아 웹사이트를 그대로 복제한 원본 환경입니다. (비교를 위한 영점 조절용)
- `/optimized-site`: AI가 제안한 개선안(Fix Plans)을 실제로 적용해 보는 최적화 실험 환경입니다.
- 두 사이트를 비교하여 LCP, TBT 등의 지표가 얼마나 향상되었는지 정량적인 성능 ROI를 도출합니다.
