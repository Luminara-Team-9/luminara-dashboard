# 🏗️ 인프라 및 서버 환경 (Infrastructure)

프로젝트를 내 컴퓨터가 아닌 실제 서버(ABRM02 로그인 서버 등)에서 띄우기 위한 **코드형 인프라(IaC) 설정 파일**들을 모아둡니다.

**구조 및 역할:**
- `/docker`: 서버를 어디서든 똑같은 환경으로 띄울 수 있게 캡슐화하는 Dockerfile 및 docker-compose 설정 파일
- `/nginx`: 트래픽을 분산하고 대시보드 웹사이트를 외부로 띄워주는 로드밸런서/웹 서버 설정
- `/lhci`: 자체 호스팅하는 Lighthouse CI 서버 구축 설정 파일
